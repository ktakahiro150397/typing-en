# 英文タイピング練習アプリ

英文タイピングゲームで勝つため、練習アプリを作成したい。
以下を要件とするが、実装順序は任意でよい。

- 英文のタイピング力向上を主目的とする。
- 英文・英単語いずれも対象とする。
- 早く打つよりも、ミスなく打てるようにすることを主目的とする。
- ユーザーが特定の単語を練習できるようにする。ワードごとに「攻略メモ」のようなものを記載できるようにする。（運指の方法など）
- システム側で、英文・英単語のうちミスしたワードなどを検知しておく。特に、キーボードの運指でミスが多い箇所の抽出を主目的とする。（W→Aの運指が苦手など）
- 苦手な運指のワードを徹底的に練習できるモードを搭載する。
- 練習対象には数字、ランダムな英数字、記号なども含める
- 練習モード中で、キーボードの入力履歴を表示できるようにする。（ミスした場合に、どのような入力が行われていたかを確認する）格闘ゲームのコマンド履歴のようなイメージです。
