# 外部レビュー ワークフロー

以下の手順で実行してください：

1. **差分の取得**:
   `git diff HEAD` を実行して現在の変更内容を取得してください。
   差分が空の場合は `git diff --cached` も試してください。

2. **レビューの実行**:
   取得した差分を `{diff}` に埋め込み、以下のコマンドを実行してください。
   ```
   copilot -p "以下のgit diffを確認し、バグや改善点があれば具体的に指摘してください。問題がなければその旨を伝えてください。\n\n<diff>\n{diff}\n</diff>" --model gpt-5.4 --no-ask-user
   ```

3. **結果の報告**:
   コマンドの出力を読み取り、指摘事項を簡潔に箇条書きでまとめてください。

** copilot CLIが応答しなかった場合はその旨を伝えてください。 **
